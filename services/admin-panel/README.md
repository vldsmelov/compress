# Admin Panel

Современная админ-панель для проверки контрактного стека. Интерфейс построен на React + Vite и даёт:

- Главную страницу с описанием логики приложения (Gateway → Aggregator → AI‑сервисы) и краткими описаниями каждого сервиса.
- Консоль Gateway для загрузки договора и просмотра HTML/JSON-ответов всех сервисов в одном месте.
- Отдельные страницы под каждый сервис с карточками эндпойнтов: можно менять путь, тело запроса, отправлять запрос и сохранять ответы.

## Запуск

```bash
cd services/admin-panel
npm install
npm run dev   # локальный дев-сервер на http://localhost:5173
npm run build # продакшен-сборка
```

### Переменные окружения
- `VITE_GATEWAY_URL` — базовый URL, который используется для всех запросов (по умолчанию `http://localhost:8099`).

## Структура экранов
- **Обзор** — документирует поток данных, показывает сколько сервисов подключено и чем занимается каждый.
- **Gateway** — загрузка файла, вывод `task_id`, предпросмотр HTML-отчётов и скачивание JSON-ответов.
- **Страница сервиса** — карточки эндпойнтов с возможностью редактировать путь/тело запроса, увидеть статус/время ответа и сохранить JSON.

## Работа с эндпойнтами
- Для GET-запросов тело запроса не отправляется.
- Тело POST/PUT/PATCH отправляется как JSON; поле «curl» помогает быстро собрать команду.
- Ответы можно скачать в JSON, чтобы приложить к отчётам или воспроизвести сценарии.

## Дизайн
- Навигация в левой колонке переключает страницы без перезагрузки состояния.
- Карточки эндпойнтов подсвечивают HTTP-метод, статус ответа и время выполнения.
- Основные цвета — глубокий тёмный sidebar и светлые панели контента для читаемости.
